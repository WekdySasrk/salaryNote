# salaryNote

## 智能记账是一款多功能记账App，前端使用uni-vue，后端使用Java，基于目前正在学习go，后续会迁移至go
功能描述：
1.**总览** 作为App首页，总览页面在可以快速记账的基础上加入了统计当日收入和总收入的可视化工作时钟，可以设置开始工作时间和下班时间以及中国特有的“单/双休”

2.**记账** 在记账页面可以对收入/支出类型进行分类，同时可以具体二级分类

3.**智能分析** 智能分析页面接入大模型，通过用户账单收集用户消费习惯进行定制化“省钱提示”，后续考虑接入各外卖+购物平台，通过用户消费习惯给用户提供定制化的消费服务
