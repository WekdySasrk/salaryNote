# 智能记账APP

一个功能完整的记账应用，基于uni-app框架开发，支持多端运行（H5、小程序、APP）。

## 功能特性

### 🏠 首页 - 实时收入计算器
- **收入设置**：设置月薪、工作天数、工作时间
- **实时收入**：每秒更新今日已赚金额
- **收入统计**：显示今年总收入和累计总收入
- **数据持久化**：关闭应用后继续累计收入

### 💰 记账功能
- **支出/收入记录**：快速记录各类收支
- **自定义分类**：支持添加、编辑、删除分类
- **智能键盘**：内置数字键盘，支持快速计算
- **详细信息**：支持备注、日期时间设置

### 📊 统计分析
- **消费统计**：按分类统计消费次数和金额
- **时间筛选**：支持今日、本周、本月、今年筛选
- **详细记录**：查看每个分类的详细消费记录
- **排序功能**：按次数或金额排序

### 🏷️ 分类管理
- **分类编辑**：添加、修改、删除自定义分类
- **图标选择**：丰富的emoji图标库
- **使用统计**：显示每个分类的使用频率
- **默认分类**：内置常用分类，不可删除

## 技术栈

- **框架**：uni-app (Vue 3)
- **构建工具**：Vite
- **样式**：SCSS + 原生CSS
- **存储**：uni.storage (本地存储)
- **UI设计**：现代化卡片式设计，支持深色模式

## 安装运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
cd uni-preset-vue-vite
npm install
```

### 开发运行
```bash
# H5端开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# APP开发
npm run dev:app
```

### 构建发布
```bash
# 构建H5
npm run build:h5

# 构建微信小程序
npm run build:mp-weixin

# 构建APP
npm run build:app
```

## 使用说明

### 1. 首次使用
1. 打开应用，进入首页
2. 点击"设置您的收入信息"
3. 填写月薪、工作天数、工作时间、开始工作日期
4. 保存设置后即可看到实时收入计算

### 2. 记账操作
1. 点击底部"记账"标签或首页快捷按钮
2. 选择支出/收入类型
3. 输入金额（可使用内置键盘）
4. 选择分类（可添加自定义分类）
5. 添加备注（可选）
6. 确认日期时间
7. 保存记录

### 3. 查看统计
1. 点击底部"统计"标签
2. 选择时间范围（今日/本周/本月/今年/全部）
3. 按次数或金额排序查看
4. 点击具体项目查看详细记录

### 4. 管理分类
1. 点击底部"分类"标签
2. 切换支出/收入分类
3. 点击"编辑"进入编辑模式
4. 添加新分类或编辑现有分类
5. 查看分类使用统计

```javascript

// 分类数据

  支出: [...],  // 支出分类列表

// 记录数据
```
